# Tutorial: BlazorVoice

**BlazorVoice** 프로젝트는 *음성 채팅*과 **AI** 기능(TTS, ChatGPT 등)을 통합하여 
웹 환경에서 실시간으로 대화하고 음성 재생을 할 수 있도록 만들어졌습니다. 
또한, *WebRTC*를 사용해 화상이나 오디오 스트리밍을 구현하고, 
**MudBlazor** UI 컴포넌트로 세련된 인터페이스를 제공합니다.
